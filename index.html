Pikadaytime

Lightweight date and time picker based on http://dbushell.github.io/Pikaday

Download this project as a .zip file Download this project as a tar.gz file

Examples

Simple date picker

Using default options. No initial date.

var picker = new Pikaday({field : document.getElementById("simple")});

Date and time picker

24 hours format with initial date. Position:right

var picker = new Pikaday({
    field          : document.getElementById("simple"),
    defaultDate    : new Date(),
    setDefaultDate : true,
    showTime       : true,
    position       : 'top right'
});
